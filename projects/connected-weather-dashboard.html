Connected Weather Dashboard with Real-Time UI

View on GitHub

Dual-core embedded weather dashboard for real-time weather, UV, and forecast data on a touch-driven display. The architecture separates networking work from UI rendering so the interface stays responsive during API calls.

My Contribution

  • Built the FreeRTOS task split between networking on Core 0 and LVGL UI on Core 1
  • Implemented HTTPS data fetches and JSON parsing with queue-based synchronization
  • Integrated SPI display and touch handling with mutex protection for the shared bus
  • Designed multi-view navigation for current weather, 24-hour forecast, and UV index rendering

Tech Stack

ESP32-S3, ESP-IDF, FreeRTOS, LVGL, C, HTTPS, cJSON, SPI Master
